Page 1: How Does the Internet Work?

Before you build websites, it's important to understand the magic behind them. The internet is a giant network of connected computers. When you visit a website, a lot happens in just a few milliseconds.

Clients and Servers

The internet works on a client-server model. Your computer's browser (like Chrome or Firefox) is the **Client**. The computer where the website is stored is the **Server**.

  • The **Client** requests information (e.g., "Give me the homepage of codewithmsmaxpro.me").
  • The **Server** receives this request, finds the correct files (like `index.html`), and sends them back to the client.

IP Addresses and DNS

Every device on the internet has a unique address called an **IP Address** (e.g., `192.168.1.1`). It's like a phone number for computers. But these numbers are hard to remember.

This is where the **DNS (Domain Name System)** comes in. The DNS is like the internet's phonebook. When you type `codewithmsmaxpro.me` into your browser, the DNS looks up this name and finds the correct IP address for the server where the website is stored. Your browser then sends its request to that IP address.
