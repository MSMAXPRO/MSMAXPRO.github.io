How Do I Even *Run* This?

This is the part that confuses beginners. You *cannot* just save this code in an `.html` file and open it in the browser (like we did with HTML/CSS). Why? Because the browser doesn't understand JSX or `import` statements.

You need a "build process" that compiles your React code into regular JavaScript. In the past, this was very hard, but today it's incredibly easy with a tool called **Vite**.

To start a real React project, you'll open your terminal and run:

# 1. This command creates a new project folder
npm create vite@latest

# 2. It will ask you for a project name (e.g., "my-react-app")
# 3. It will ask you to pick a framework. Choose "React".
# 4. Then, just follow its instructions:
cd my-react-app
npm install
npm run dev

This will start a local server, and you can start building your components for real!

Conclusion: Thinking in React

You now understand the "big ideas" of React. The workflow is:

  1. Break your UI into small, reusable **Components**.
  2. Write the UI for those components using **JSX**.
  3. Pass data *down* from parents to children using **Props**.
  4. Give your components their own *memory* using **State**.
