Real-time Chat App

Build a full-stack chat application where users can join rooms and send messages instantly using WebSockets.

üöÄ The Project

You've built simple CRUD apps in tutorials, but now it's time to see the power of real-time communication! In this project, we'll build a chat app that works like Slack or Discord. Users will be able to join a room, and their messages will instantly appear for other users *without* a page reload.

This project will introduce you to the magic of WebSockets and explain how modern web apps handle real-time data.

‚ú® Key Features

  • Users can enter a username and a "Room ID".
  • Multiple users can join the same room.
  • Messages sent by one user are broadcasted instantly to all other users in that room.
  • A "User has joined" or "User has left" notification will be displayed.

Part 1: Backend Setup (Node.js & Socket.io)

  1. Set up a basic Node.js and Express server.
  2. Install and integrate the `socket.io` library.
  3. Listen for the Socket.io 'connection' event.
  4. Create a 'join_room' event. When a user joins, add them to the room using the `socket.join(room)` method.
  5. Create a 'send_message' event. When a message is received, send it to all users in that room using `io.to(room).emit('receive_message', ...)`
  6. Handle the 'disconnect' event to send notifications when users leave.

Part 2: Frontend Setup (React)

  1. Create a new React app.
  2. Install the `socket.io-client` library.
  3. Create a 'Home' component where the user will enter their name and room ID.
  4. Create a 'Chat' component that will connect to the backend using `socket.io-client`.
  5. Use the `useEffect` hook to listen for the 'receive_message' event and add new messages to a state array (`messages`).
  6. Create an input form that will emit the 'send_message' event when the user sends a message.
  7. Map over the messages array to render them on the screen.
