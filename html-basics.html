Chapter 2: HTML - The Structure

Frontend Developer Roadmap

What is HTML?

HTML stands for Hypertext Markup Language. It is the standard language used to create and design the content on a web page. Think of it as the skeleton of a house. Just as a skeleton provides the basic structure for a house, HTML provides the basic structure for a website. It tells the browser what content to display and how to organize itâ€”where the headings are, where the paragraphs go, and where to put images.

Basic Structure of an HTML Document

Every HTML document follows a standard structure. This structure tells the browser that it is reading an HTML file and helps it understand the content correctly.

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>My First Heading</h1>
        <p>My first paragraph.</p>
    </body>
</html>
  • <!DOCTYPE html>: This declaration defines that this document is an HTML5 document.
  • <html>: This is the root element of an HTML page. All other elements go inside it.
  • <head>: This element contains meta-information about the page, such as the title, links to CSS files, and character set. This information is not displayed on the page itself.
  • <title>: This specifies the title for the page, which is shown in the browser's title bar or in the page's tab.
  • <body>: This element contains all the visible content of a webpage, such as headings, paragraphs, images, links, etc.

Common HTML Tags

HTML uses "tags" to define different types of content. Most tags come in pairs: an opening tag (e.g., <p>) and a closing tag (e.g., </p>). Here are some of the most common tags you will use every day:

Headings and Paragraphs

Headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading, and <h6> defines the least important. Paragraphs are defined with the <p> tag.

Links (Anchors)

Links are one of the most important parts of the web. They allow you to navigate from one page to another. The <a> tag (anchor tag) is used to create a link. The destination of the link is specified in the href attribute.

<a href="https://www.google.com">Visit Google</a>

Images

To display an image on a page, you use the <img> tag. This is a self-closing tag and does not have a closing tag. The path to the image is specified in the src attribute, and it's very important to provide an alt attribute, which describes the image for screen readers and if the image fails to load.

<img src="images/logo.png" alt="Website Logo">

Lists

HTML provides two types of lists:

  • Unordered Lists (<ul>): Used for items where the order doesn't matter. Each list item starts with the <li> tag.
  • Ordered Lists (<ol>): Used for items where the order is important. Each list item also starts with the <li> tag.

Semantic HTML

As you get more advanced, it's important to use "semantic" HTML. This means using tags that clearly describe their content. For example, instead of using <div> for everything, you should use tags like:

  • <header>: For the introductory content or navigation links.
  • <footer>: For the footer section of a page.
  • <nav>: For a set of navigation links.
  • <section>: For a thematic grouping of content.
  • <article>: For self-contained content like a blog post.

Using semantic tags makes your code easier to read and improves your website's SEO (Search Engine Optimization), helping search engines like Google understand your content better.
