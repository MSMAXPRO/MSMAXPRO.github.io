E-commerce Cart

Build a functional front-end for a product page with a working "Add to Cart" and "View Cart" feature using React Context API.

üöÄ The Project

This project is essential for every React developer's portfolio. We will build a professional-looking product page. When a user clicks 'Add to Cart', the item will be added to a global cart state. The user can then navigate to the `/cart` page to see all their items.

The main focus of this project is Global State Management. We will use the React Context API to avoid "prop drilling" (passing props down multiple levels).

‚ú® Key Features

  • A product page showing item details.
  • "Add to Cart" button to add items to a global cart.
  • A separate "Cart" page (`/cart`) to display all selected items.
  • Ability to see the total number of items in the cart (e.g., in the navbar).
  • (Optional) Ability to increase/decrease item quantity in the cart.

We will use these modern technologies:

üõ†Ô∏è Step-by-Step Guide

Part 1: Setup and Routing

  1. Create a new React app and install `react-router-dom`.
  2. Set up basic routing in `App.js`: `/` (Products page) and `/cart` (Cart page).
  3. Create a Navbar component with links to both pages.

Part 2: The Context API Magic

  1. Create a new folder `context` and a `CartContext.js` file inside it.
  2. Create `CartContext` using `createContext`.
  3. Create a `CartProvider` component that accepts `children`.
  4. Inside `CartProvider`, manage the `cartItems` state (using `useState`).
  5. Create functions: `addToCart(item)`, `removeFromCart(id)`.
  6. Pass these functions and the `cartItems` state in the context's `value`.
  7. Wrap your entire `App` component with `CartProvider` in `index.js`.

Part 3: Building the Pages

  1. Products Page: Create an array of dummy product data. Map over the data and display it. Add an "Add to Cart" button to each product.
  2. On the button's `onClick`, call the `addToCart` function from `useContext(CartContext)`.
  3. Cart Page: Access the `cartItems` array using `useContext`.
  4. Map over the array to display a list of all cart items. Calculate and show the total price.
  5. Navbar: Access `cartItems.length` from `useContext` and display it, e.g., "Cart (3)".

üî• Next Steps & Challenges

After completing the project, try these ideas:

  • Use `useReducer`: For more complex state, use the `useReducer` hook instead of `useState` in your context.
  • **Quantity:** Add a feature to increase or decrease item quantity on the cart page.
  • **Local Storage:** Save the cart items to `localStorage` so the cart isn't empty after a page reload.
  • **Backend:** Fetch products from a fake API (like `fakestoreapi.com`).
