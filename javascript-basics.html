Frontend Developer Roadmap

Welcome to the most exciting chapter! If HTML is the structure (skeleton) and CSS is the style (soul), then JavaScript (JS) is the website's brain. It is a proper programming language that makes your website interactive. Without JavaScript, the web would just be static, boring documents. JavaScript is the power that does all of this:

  • Image sliders and pop-up windows.
  • Form validation (checking if a user entered a valid email).
  • Changing content on the page without reloading.
  • Creating games, fetching data, and so much more!

In this guide, we will understand the core concepts of JavaScript from scratch, so you can add real interactivity to your website.

    Section 4: Working with Data Structures

    In real-world applications, data isn't simple. We store data in lists (Arrays) or collections (Objects).

    Arrays (Lists)

    An array is a single variable that can hold multiple values. It is zero-indexed (counting starts from 0).

    const fruits = ["Apple", "Banana", "Mango"];

// Accessing items
console.log(fruits[0]); // Output: Apple
console.log(fruits[1]); // Output: Banana

// Adding a new item
fruits.push("Orange");
console.log(fruits); // Output: ["Apple", "Banana", "Mango", "Orange"]

// Looping through an array
fruits.forEach(function(fruit) {
    console.log("I like " + fruit);
});

    Objects (Collections)

    Objects let you store related data in key-value pairs. They are perfect for representing real-world things.

    const user = {
    firstName: "MSMAX",
    lastName: "PRO",
    isStudent: true,
    age: 20,
    sayHello: function() {
        console.log("Hello!");
    }
};

// Accessing properties (Dot Notation)
console.log(user.firstName); // Output: MSMAX
console.log(user.age); // Output: 20

// Calling a method (function inside an object)
user.sayHello(); // Output: Hello!
